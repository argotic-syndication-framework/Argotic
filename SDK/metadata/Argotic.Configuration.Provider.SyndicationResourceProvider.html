ApplicationName

Gets or sets the name of the application using the custom syndication resource provider.

Declaration
public abstract string ApplicationName { get; set; }
Property Value
Type Description
System.String

The name of the application using the custom syndication resource provider.
Remarks

The ApplicationName property value is stored in the data source with related syndication resource information to associate a resource with a particular application.

Because syndication resource providers store resource information uniquely for each application, multiple applications can use the same data source without running into a conflict if duplicate syndication resources are created. Alternatively, multiple applications can use the same syndication resource data source by specifying the same ApplicationName.

In your syndication resource provider implementation, you will need to ensure that your data schema includes the ApplicationName and that data source queries and updates also include the ApplicationName.

GetResources(Int32, Int32, out Int32)

Gets a collection of all the resources in the data source in pages of data.

Parameters

System.Int32 pageSize

The size of the page of results to return.
System.Int32 totalRecords

The total number of matched resources.
Returns
Type Description
System.Collections.ObjectModel.Collection<ISyndicationResource>

A System.Collections.ObjectModel.Collection<T> that contains a page of ISyndicationResource objects with a size of pageSize, beginning at the page specified by pageIndex.
Remarks

GetResources(Int32, Int32, out Int32) returns a list of all of the resources from the data source for the configured ApplicationName property. Syndication resources are returned in order of last time they were updated in the data source.

The results returned by GetResources(Int32, Int32, out Int32) are constrained by the pageIndex and pageSize parameters. The pageSize parameter identifies the number of ISyndicationResource objects to return in the collection. The pageIndex parameter identifies which page of results to return, where 0 identifies the first page. The totalRecords parameter is an out parameter that is set to the total number of syndication resources in the data source.

For example, if there are 13 resources in the data source, and the pageIndex value was 1 with a pageSize of 5, then the System.Collections.ObjectModel.Collection<T> would contain the sixth through the tenth resources returned. The totalRecords parameter would be set to 13.

Exceptions
Type Condition
System.ArgumentOutOfRangeException

The pageIndex is less than zero.
System.ArgumentOutOfRangeException

The pageSize is less than or equal to zero.
