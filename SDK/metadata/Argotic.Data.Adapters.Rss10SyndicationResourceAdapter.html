Remarks

The Rss10SyndicationResourceAdapter serves as a bridge between a RssFeed and an XML data source. The Rss10SyndicationResourceAdapter provides this bridge by mapping Fill(RssFeed), which changes the data in the RssFeed to match the data in the data source.

This syndication resource adapter is designed to fill RssFeed objects using a System.Xml.XPath.XPathNavigator that represents XML data that conforms to the RSS 1.0 specification.
