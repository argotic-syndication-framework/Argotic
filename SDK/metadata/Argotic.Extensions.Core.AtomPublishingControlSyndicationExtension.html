[Serializable]
public class AtomPublishingControlSyndicationExtension : SyndicationExtension, ISyndicationExtension, IXmlSerializable, IComparable
Remarks

The AtomPublishingControlSyndicationExtension extends syndicated content to specify publishing control of published resources. This syndication extension conforms to the Atom Publishing Protocol 1.0 specification, which can be found at http://bitworking.org/projects/atom/rfc5023.html.

The AtomPublishingControlSyndicationExtension class implements the app:control element of the Atom Publishing Protocol.

The AtomPublishingControlSyndicationExtension may appear as a child of an Atom Entry that is being created or updated via the Atom Publishing Protocol. The AtomPublishingControlSyndicationExtension must appear only once in an Atom Entry.

A AtomPublishingControlSyndicationExtensionContext object that contains information associated with the current syndication extension.
Remarks

The Context encapsulates all of the syndication extension information that can be retrieved or written to an extended syndication entity. Its purpose is to prevent property naming collisions between the base SyndicationExtension class and any custom properties that are defined for the custom syndication extension.
