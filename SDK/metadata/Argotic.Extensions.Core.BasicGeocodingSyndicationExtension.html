public class BasicGeocodingSyndicationExtension : SyndicationExtension, ISyndicationExtension, IXmlSerializable, IComparable
Remarks

The BasicGeocodingSyndicationExtension extends syndicated content to specify a basic RDF vocabulary that provides the Semantic Web community with a namespace for representing latitude, longitude and other information about spatially-located things. This syndication extension conforms to the Basic Geo (WGS84 lat/long) Vocabulary specification, which can be found at http://www.w3.org/2003/01/geo/.

BasicGeocodingSyndicationExtensionContext

A BasicGeocodingSyndicationExtensionContext object that contains information associated with the current syndication extension.
Remarks

The Context encapsulates all of the syndication extension information that can be retrieved or written to an extended syndication entity. Its purpose is to prevent property naming collisions between the base SyndicationExtension class and any custom properties that are defined for the custom syndication extension.
