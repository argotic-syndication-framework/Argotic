Syntax
[Serializable]
public class CreativeCommonsSyndicationExtension : SyndicationExtension, ISyndicationExtension, IXmlSerializable, IComparable
Remarks

The CreativeCommonsSyndicationExtension extends syndicated content to specify which Creative Commons license is applicable. This syndication extension conforms to the creativeCommons RSS Module 1.0 specification, which can be found at http://backend.userland.com/creativeCommonsRssModule.

Property Value
Type Description
CreativeCommonsSyndicationExtensionContext

A CreativeCommonsSyndicationExtensionContext object that contains information associated with the current syndication extension.
Remarks

The Context encapsulates all of the syndication extension information that can be retrieved or written to an extended syndication entity. Its purpose is to prevent property naming collisions between the base SyndicationExtension class and any custom properties that are defined for the custom syndication extension.
