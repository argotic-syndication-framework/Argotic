Enum DublinCoreTypeVocabularies

Represents a general, cross-domain list of approved terms that may be used as values when identifying the nature or genre of a resource.

Namespace: Argotic.Extensions.Core
Assembly: Argotic.Extensions.dll

For more information about the DCMI Type Vocabulary, see http://dublincore.org/documents/dcmi-type-vocabulary/.
