[Serializable]
public class ITunesSyndicationExtension : SyndicationExtension, ISyndicationExtension, IXmlSerializable, IComparable
Remarks

The ITunesSyndicationExtension extends syndicated content to specify iTunes podcasting information. This syndication extension conforms to the iTunes RSS Tags 1.0 specification, which can be found at http://www.apple.com/itunes/store/podcaststechspecs.html#rss.

Property Value
Type Description
ITunesSyndicationExtensionContext

A ITunesSyndicationExtensionContext object that contains information associated with the current syndication extension.
Remarks

The Context encapsulates all of the syndication extension information that can be retrieved or written to an extended syndication entity. Its purpose is to prevent property naming collisions between the base SyndicationExtension class and any custom properties that are defined for the custom syndication extension.
