Syntax
public class SiteSummaryUpdateSyndicationExtension : SyndicationExtension, ISyndicationExtension, IXmlSerializable, IComparable
Remarks

The SiteSummaryUpdateSyndicationExtension extends syndicated content to specify hints to aggregators and other entities regarding how often a feed is updated. This syndication extension conforms to the RDF Site Summary 1.0 Modules: Syndication 1.0 specification, which can be found at http://web.resource.org/rss/1.0/modules/syndication/.

SiteSummaryUpdateSyndicationExtensionContext

A SiteSummaryUpdateSyndicationExtensionContext object that contains information associated with the current syndication extension.
Remarks

The Context encapsulates all of the syndication extension information that can be retrieved or written to an extended syndication entity. Its purpose is to prevent property naming collisions between the base SyndicationExtension class and any custom properties that are defined for the custom syndication extension.
