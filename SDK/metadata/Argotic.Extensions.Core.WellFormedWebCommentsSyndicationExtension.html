Syntax
public class WellFormedWebCommentsSyndicationExtension : SyndicationExtension, ISyndicationExtension, IXmlSerializable, IComparable
Remarks

The WellFormedWebCommentsSyndicationExtension extends syndicated content to specify the human and machine readable endpoints for users to provide commentary on published content. This syndication extension conforms to the Well-Formed Web Comment API 1.0 specification, which can be found at http://wellformedweb.org/news/wfw_namespace_elements/.

WellFormedWebCommentsSyndicationExtensionContext

A WellFormedWebCommentsSyndicationExtensionContext object that contains information associated with the current syndication extension.
Remarks

The Context encapsulates all of the syndication extension information that can be retrieved or written to an extended syndication entity. Its purpose is to prevent property naming collisions between the base SyndicationExtension class and any custom properties that are defined for the custom syndication extension.
