Class YahooMediaRestriction

Represents a means of allowing restrictions to be placed on the aggregator rendering the media in the feed.

Remarks

Currently, restrictions are based on distributor or country code. A YahooMediaRestriction is purely informational and no obligation can be assumed or implied. Only one YahooMediaRestriction object of the same type can be applied to a media object, all others will be ignored.

Entities

Gets the entities this restriction applies to.

Declaration
public Collection<string> Entities { get; }
Property Value
Type Description
System.Collections.ObjectModel.Collection<System.String>

A System.Collections.ObjectModel.Collection<T> collection of System.String objects that represent the entities this restriction applies to. The default value is an empty collection.
Remarks

To allow a producer to explicitly declare their intentions, two literal entity names are reserved: all and none. These literals can only be used once.

When the restriction EntityType is Uri the elements in this collection should represent distributor System.Uri.

When the restriction EntityType is Country the elements in this collection should represent country codes. See ISO 3166 for a listing of the permissible country codes.
