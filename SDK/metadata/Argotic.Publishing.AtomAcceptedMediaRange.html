Class AtomAcceptedMediaRange

Represents a media range as defined in RFC 2616: Hypertext Transfer Protocol that specifies a type of representation that can be added to a AtomMemberResources.

Remarks

The AtomAcceptedMediaRange class implements the app:accept element of the Atom Publishing Protocol.

The content value of the MediaRange property for an AtomAcceptedMediaRange is a media range as defined in RFC 2616. The media range specifies a type of representation that can be added to a AtomMemberResources via a POST operation.

The AtomAcceptedMediaRange is similar to the HTTP Accept request-header [RFC 2616]. Media type parameters are allowed within AtomAcceptedMediaRange, but AtomAcceptedMediaRange has no notion of preference e.g. accept-params or q arguments, as specified in section 14.1 of RFC 2616 are not significant.

See http://www.iana.org/assignments/media-types for a listing of the registered IANA MIME media types and sub-types.
