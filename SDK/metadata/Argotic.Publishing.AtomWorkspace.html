Class AtomWorkspace

Represents a server-defined group of AtomMemberResources objects that describe member resources.

Remarks

The AtomWorkspace class implements the app:workspace element of the Atom Publishing Protocol.

A AtomServiceDocument groups AtomMemberResources into AtomWorkspace. Operations on AtomWorkspace, such as creation or deletion, are not defined by the Atom Publishing Protocol specification. The Atom Publishing Protocol specification assigns no meaning to AtomWorkspace; that is, a AtomWorkspace does not imply any specific processing assumptions.

There is no requirement that a server support multiple AtomWorkspace. In addition, a AtomMemberResources may appear in more than one AtomWorkspace.
