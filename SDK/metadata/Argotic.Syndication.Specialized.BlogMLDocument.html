[Serializable]
public class BlogMLDocument : ISyndicationResource, IExtensibleSyndicationObject
Remarks

This implementation conforms to the BlogML 2.0 specification, which can be found at http://blogml.org.

The purpose of this format is to provide an open format derived from XML to store and restore the content of a blog.

Returns
Type Description
ISyndicationExtension

The first syndication extension that matches the conditions defined by the specified predicate, if found; otherwise, the default value for ISyndicationExtension.
Remarks

The System.Predicate<T> is a delegate to a method that returns true if the object passed to it matches the conditions defined in the delegate. The elements of the current Extensions are individually passed to the System.Predicate<T> delegate, moving forward in the Extensions, starting with the first element and ending with the last element. Processing is stopped when a match is found.

To receive notification when the operation has completed or the operation has been canceled, add an event handler to the Loaded event. You can cancel a LoadAsync(Uri, SyndicationResourceLoadSettings, ICredentials, IWebProxy, Object) operation by calling the LoadAsyncCancel() method.

After calling LoadAsync(Uri, SyndicationResourceLoadSettings, ICredentials, IWebProxy, Object), you must wait for the load operation to complete before attempting to load the syndication resource using the LoadAsync(Uri, Object) method.
